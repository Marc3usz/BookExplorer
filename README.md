#Book Explorer
##Jak odpalić

- pull na repo

- otworzyć w android studio

- gradle sync

- uruchomić na maszynie wirtualnej android z API 35+

-  ## UWAGA - jesli tak nie zadziala proszę kminić dalej ale szczerzę to już chyba coś się zwaliło u was bo u mnie działa

 - modelView

- stateFlow

- DataStore

- Retrofit

- Jetpack Compose

- Navigation Compose

- LazyColumn

- PullToRefresh

- Repository pattern

- Material3 do icon/stylowania

- Dark Mode

- Paginacja

- Pisane w kotlinie 2.1.0

- Animacje (takie fade'y i przy klikaniu chyba taki wave niby defaultowy ale jest)
